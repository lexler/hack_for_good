## Keyboard Shortcuts

You can use your keyboard instead of tapping:

**Number Pad Layout:**
```
7 = TA    8 = BD    9 = RF
4 = LP    5 = UNDO  6 = UP  
1 = QU    2 = CM    3 = NTA
```

**QWERTY Layout (Alternative):**
```
Q = TA    W = BD    E = RF
A = LP    S = UNDO  D = UP
Z = QU    X = CM    C = NTA
```
